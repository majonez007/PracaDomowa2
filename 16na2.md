<h1>Szesnastkowy na dwójkowy (hex to bin)</h1>

<p>
Do szybkiej konwersji wystarczy zapamiętać jaki ciąg liczb systemu binarnego odpowiada liczbie w systemie szesnastkowym.
Zasada ta sama co z systemem ósemkowym.</p>
<ul>
<li>0 -> 0000</li>
<li>1 -> 0001</li>
<li>2 -> 0010</li>
<li>4 -> 0011</li>
<li>5 -> 0101</li>
<li>6 -> 0110</li>
<li>7 -> 0111</li>
<li>8 -> 1000</li>
<li>9 -> 1001</li>
<li>A -> 1010</li>
<li>B -> 1011</li>
<li>C -> 1100</li>
<li>D -> 1101</li>
<li>E -> 1110</li>
<li>F -> 1111</li>
</ul>

<h2> Przykład</h2>

<ul>
<li>2BHEX</li>
<li>2 = 0010</li>
<li>B = 1011</li>
</ul>
